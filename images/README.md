# Dossier Images

Ce dossier contient les images utilisées dans le portfolio.

## Structure recommandée

```
images/
├── projects/          # Captures d'écran de vos projets
│   ├── projet1.jpg
│   ├── projet2.jpg
│   └── projet3.jpg
├── profile.jpg        # Votre photo de profil
├── favicon.ico        # Icône du site
└── social-preview.png # Image pour partage sur réseaux sociaux
```

## Format recommandé

- **Photos de profil** : 400x400px, format JPG ou PNG
- **Projets** : 800x600px minimum, format JPG optimisé
- **Social preview** : 1200x630px, format PNG ou JPG

## Optimisation

Utilisez des outils comme [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/) pour compresser vos images avant de les ajouter.
